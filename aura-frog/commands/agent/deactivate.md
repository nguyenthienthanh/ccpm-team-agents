# Command: agent:deactivate

**Category:** Agent Management  
**Syntax:** `agent:deactivate <agent-name>`

---

## Description

Deactivate a specific agent from the current workflow.

---

## Usage

```
agent:deactivate ui-designer
```

---

## Output

```markdown
⏸️  Agent Deactivated: ui-designer

The UI Designer agent has been deactivated and will no longer participate in the current workflow.

You can reactivate it anytime with:
agent:activate ui-designer
```

**Version:** 1.0.0
